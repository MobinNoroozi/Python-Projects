# Python Projects

This repository contains a collection of Python projects I built while practicing core programming concepts, problem-solving, object-oriented programming, and game/application logic. Each project is organized in its own folder with a "DayX" prefix indicating the learning sequence.

## Projects Included

| Day | Project | Description |
|-----|---------|-------------|
| Day 6 | Password Generator Terminal | Terminal-based password generator with customizable options |
| Day 7 | Hangman Game | Classic word-guessing game with ASCII art |
| Day 8 | Caesar Cipher | Encryption and decryption using the Caesar cipher algorithm |
| Day 9 | Secret Auction | Blind auction program with bidder management |
| Day 10 | Blackjack Game | Casino card game with dealer logic |
| Day 11 | Number Guessing Game | Interactive number guessing game with difficulty levels |
| Day 12 | Higher Lower Game | Comparison game with Instagram followers data |
| Day 13 | Coffee Machine Program | Procedural coffee machine simulator |
| Day 14 | Coffee Machine Program - OOP | Object-oriented version of the coffee machine |
| Day 17 | Quiz Game | Quiz application with question database and scoring |
| Day 18 | Turtle Dot Spot Painting | Graphics-based dot painting using Turtle graphics |
| Day 19 | Turtle Racing Game | Racing game with turtle graphics |
| Day 20 | Snake Game | Classic Snake game with collision detection |
| Day 21 | Pong Game | Two-player Pong game implementation |
| Day 22 | Turtle Crossing Game | Frogger-style crossing game with Turtle graphics |
| Day 23 | Mail Merging & File/Directory Practice | File manipulation and directory operations |
| Day 24 | State Guessing Game (CSV & Files) | Geography game using CSV data and file handling |

## Skills Demonstrated

- Python fundamentals (variables, loops, conditionals, functions)
- Object-oriented programming (classes, inheritance, encapsulation)
- File and folder organization
- Game logic and mechanics
- User input handling and validation
- Data structures (lists, dictionaries, tuples)
- File I/O and CSV handling
- Graphics programming with Turtle
- Debugging and project structure
- Algorithm implementation

## Repository Structure

Each project folder contains:
- Main Python script(s)
- Supporting files (data files, assets, etc.)
- README or comments explaining the project
